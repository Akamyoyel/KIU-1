오늘의 수업은 깃허브에 대해 알아보고
지역저장소와 원격저장소를 연결하는 수업입니다.
깃허브는 원격 저장소의 일종으로, 개발자들이 자주사용하며, 오픈소스가 많습니다.
깃허브를 가입한후에 지역저장소에서 문서를 작성한후에 
git add 파일이름으로 커밋을 한다음 깃허브의 주소를 확인합니다.
이후 git remote add origin 주소 를 입력하여 원격 저장소를 연결한후
git remote -v로 확인합니다.
그다음, git push -u origin master로 지역저장소의 소스를
원격저장소에 업로드합니다.
만약 오류가 발생할경우 git pull origin master로 깃허브의 저장내용을 불러와서 
다시 git push를 해봅니다.
